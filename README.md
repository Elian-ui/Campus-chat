# Campus Chat App

This is a mobile chat application built using Expo for students on campus to communicate and collaborate.

## Getting Started

To get started with the Campus Chat App, follow these steps:

### Prerequisites

- Node.js and npm installed on your machine.
- Expo CLI installed globally. You can install it by running:
  npm install -g expo-cli

### Installation

1. Clone the repository to your local machine:
   git clone <repository-url>

2. Navigate to the project directory:
   cd campus-chat-app

3. Install project dependencies:
   npm install

### Starting the Development Server

To start the Expo development server, run:
npm start

### Running on Android

1. After starting the development server, a QR code will be displayed in the terminal and Expo Dev Tools in the browser.
2. To run the app on an Android device:

   - Install the Expo Go app from the Google Play Store.
   - Scan the QR code using the Expo Go app.
   - The app will be loaded on your device.

3. To run the app on an Android emulator:
   - Open Android Studio and start an Android emulator.
   - In the Expo Dev Tools, click on "Run on Android device/emulator".
   - Expo CLI will build the app and load it into the emulator.

### Testing

Once the app is loaded on your Android device or emulator, you can interact with it, test its features, and verify that everything is working as expected.

## Features

- Real-time chat functionality.
- User authentication and registration.
- Group chat and private messaging.
- File and image sharing.
- Notifications for new messages.
